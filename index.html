Sobre Mim

Sou um desenvolvedor web com experiência em criar websites funcionais e visualmente atraentes. Tenho paixão por aprender novas tecnologias e aprimorar minhas habilidades. Possuo um sólido conhecimento em HTML, JavaScript e WordPress, o que me permite criar soluções personalizadas e dinâmicas para diversas necessidades.

Estou sempre ligado nas últimas tendências e melhores práticas de desenvolvimento web, participando de cursos, workshops e comunidades online. Meu foco é entregar trabalho de alta qualidade, com atenção aos detalhes e à satisfação do cliente.

Projetos

Projeto cordel

Meu projeto mais recente é o "Cordel", um site responsivo dedicado a celebrar a rica tradição de cordel. Este site oferece uma experiência imersiva, trazendo a poesia popular nordestina para o mundo digital de forma vibrante e acessível.

A página foi cuidadosamente projetada para ser totalmente responsiva, garantindo uma navegação fluida e agradável em qualquer dispositivo, seja desktop, tablet ou smartphone. Utilizando HTML e CSS, o site apresenta um design visualmente atraente, com elementos interativos que engajam os visitantes e os convidam a explorar mais.

Projeto Curiosidade de tecnologia

Curiosidades de tecnologia é um site dinâmico e responsivo focado em curiosidades sobre tecnologias, com um destaque especial para o universo Android. Criado com HTML e CSS, este site é projetado para ser acessível e atraente em qualquer dispositivo, proporcionando uma experiência de navegação agradável para todos os usuários.

A pagina é repleto de links interativos que levam os visitantes a explorar uma variedade de tópicos fascinantes sobre o assunto. O site é cuidadosamente estruturada para facilitar a leitura e a interação, mantendo os usuários engajados e curiosos.
